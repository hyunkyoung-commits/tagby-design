# 프로젝트 개요

## 목적
TAGby Design 디지털 마케팅 대행사 웹사이트. 회사 소개, 서비스 안내, 포트폴리오, 팀 소개 페이지로 구성된 정적 HTML 사이트.

## 배포
- GitHub 저장소: https://github.com/hyunkyoung-commits/tagby-design
- Vercel 자동 배포 (main 브랜치 push 시 자동 배포)
