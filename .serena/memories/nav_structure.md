# 네비게이션 구조

## 상단 네비게이션 메뉴
1. **디지털마케팅** (dropdown)
   - 퍼포먼스 마케팅 → performance-marketing.html
   - 인플루언서 마케팅 → influencer-marketing.html

2. **포트폴리오** (mega menu - 3컬럼)
   - 패션: 스파오, 스파오키즈, 로엠, 믹스코, 뉴발란스
   - 생활/잡화: 코코도르
   - 모바일앱: 민병철유폰

3. **팀소개** (dropdown)
   - 퍼포먼스팀 → team-performance.html
   - 디지털마케팅팀 → team-digital.html
   - 디자인팀 → team-design.html

4. **케이스스터디** (dropdown)
   - 디자인 스터디 → design-study.html

## 네비게이션 종류
- **밝은 배경 nav** (index.html, portfolio.html 등): 텍스트 색상 `text-gray-700`
- **어두운 배경 nav** (포트폴리오 상세 페이지): 텍스트 색상 흰색/반투명
